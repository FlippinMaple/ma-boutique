## 2. Statut des décisions

### Validé

- voix directe, calme, concise et assurée ;
- refus du ton corporate ;
- refus du faux slang ;
- refus des clichés canadiens ;
- refus du discours artisanal inventé ;
- aucune promesse non vérifiée ;
- aucune histoire de fabrication inventée ;
- produit et fonction toujours compréhensibles ;
- écriture accessible ;
- **français québécois naturel** = langue source et langue principale de la marque et du site ;
- **tutoiement** = registre principal (direct, mature, accessible, cohérent ; jamais adolescent, caricatural, trop familier ou rempli de slang) ;
- **anglais canadien** offert comme version complète et adaptée naturellement (jamais traduction mot à mot) ;
- **une seule langue affichée à la fois** dans les contenus principaux du site ;
- **`NO FIXED LINE.`** demeure en anglais dans les versions française et anglaise ; aucune traduction automatique ;
- **cohérence du registre** dans une même page, une même surface ou un même échange ;
- service client : tutoiement par défaut ; adaptation possible au registre du client sans alternance arbitraire `tu` / `vous` ;
- français naturel au Québec sans caricature ;
- anglais canadien adapté (même personnalité, maturité et retenue) ;
- signature utilisée avec retenue ;
- voix légèrement irrévérencieuse, mais jamais insolente, agressive ou adolescente ;
- POD = moyen de production, jamais sujet identitaire ;
- un seul grand usage dominant de **`NO FIXED LINE.`** sur une page d’accueil.

### Provisoire

- signature **`NO FIXED LINE.`** (usage de travail verbal) ;
- intensité du ton brut ;
- intensité exacte de l’irrévérence et contextes où elle peut apparaître ;
- fréquence des fragments ;
- niveau d’usage des capitales ;
- vocabulaire final des collections ;
- certains CTA ;
- titres de campagne ;
- orthographe canadienne anglaise exacte (ex. colour) à figer en rédaction.

### Ouvert

- validation juridique de la signature ;
- mécanisme technique du sélecteur de langue ;
- méthode de mémorisation de la préférence linguistique ;
- architecture technique d’internationalisation (routes, URLs, SEO bilingue détaillé) ;
- textes finaux français et anglais approuvés ;
- noms de collections et de produits ;
- textes définitifs de la page d’accueil ;
- politique complète du service client ;
- politique éditoriale réseaux sociaux ;
- ton des collaborations ;
- termes propriétaires éventuels.

## 3. Essence verbale

**Essence :** Flippin’ Maple parle comme une marque indépendante qui avance — claire, calme, précise — sans inventer ce qu’elle n’est pas.

**Principes directeurs :**

1. Aller droit au point.
2. Laisser de l’espace.
3. Évoquer le mouvement sans le surjouer.
4. Parler comme une marque, pas comme une publicité.
5. Ne jamais inventer ce qui n’est pas vrai.

## 6. Ton principal

- phrases courtes à moyennes ;
- rythme mesuré ;
- ponctuation sobre (peu d’exclamations) ;
- densité faible à moyenne ;
- fragments OK en campagne / titres / déclaration ;
- capitales pour libellés courts seulement ;
- impératifs utiles pour CTA, pas pour harceler ;
- questions rares, jamais rhétoriques vagues ;
- humour rare et sec ;
- émotion contenue ;
- tutoiement (**Validé**) : direct, mature, accessible, cohérent ; proximité sans familiarité caricaturale.

**Contractions :**

- permises : formes courantes comme `c’est`, `on` ;
- évitées dans l’UX, les produits, les courriels transactionnels et le service client : formes très orales comme `t’es` ;
- `t’es` possible seulement dans une campagne volontairement familière et **approuvée** ;
- aucun registre oral forcé.

Fragments interdits dès que la clarté est critique : instructions, formulaires, panier, paiement, politiques, service client.

## 8. Français de marque

- naturel au Québec, professionnel, contemporain, accessible ;
- non caricatural, non « internationalisé » artificiellement, non calqué sur l’anglais.

| Sujet | Direction de travail |
|---|---|
| Tutoiement / vouvoiement | **Tutoiement** = registre principal (**Validé**) : proximité naturelle pour une marque indépendante ; cohérence avec une boutique destinée directement au consommateur ; ton accessible sans familiarité excessive ; jamais adolescent, caricatural, trop familier ou rempli de slang. Le service client utilise normalement le tutoiement ; il peut s’adapter au registre employé par le client lorsque cela améliore l’échange ; ne jamais alterner `tu` et `vous` dans un même échange ou une même surface. |
| Cohérence | Un registre cohérent par page, surface ou échange (**Validé**) ; ne pas mélanger tu/vous dans la même interaction |
| Contractions | `c’est`, `on` : permis. `t’es` et oral très familier : évités hors campagne familière approuvée. Aucun registre oral forcé. |
| Anglicismes OK | Termes produit établis si clairs (hoodie si nécessaire → préférer « coton ouaté » quand pertinent) |
| Anglicismes à éviter | Slang importé, CTA anglais dans UI française (« Let’s go ») |
| Ponctuation | Française ; espaces avant `?` `!` `;` `:` selon usage soigné |
| Capitalisation | Titres sobres ; pas de TOUT EN CAPITALES dans les paragraphes |
| Nombres / devises | `$` avec montants clairs (ex. `48,00 $`) |
| Dates | Format local clair (ex. 21 juillet 2026) |
| Apostrophes | Typographiques (`’`) |

## 10. Stratégie bilingue

**Validé** comme direction de travail : **français québécois source et principal + anglais canadien complet adapté**.

| Règle | Statut |
|---|---|
| Français = langue source et principale | **Validé** |
| Anglais = version complète adaptée (pas calque) | **Validé** |
| Une seule langue affichée à la fois dans les contenus principaux | **Validé** |
| Tutoiement dans la version française | **Validé** |
| `NO FIXED LINE.` en anglais dans les deux versions | **Validé** |
| Sélecteur de langue clair (éventuel) | **Ouvert / hors portée technique** |

| Traduction littérale FR → EN | Calque ; perte de personnalité |

## 11. Signature `NO FIXED LINE.`

| Champ | Contenu |
|---|---|
| Sens | Pas de trajectoire imposée ; liberté de changer de direction ; mouvement ; autonomie |
| Intention | Attitude, pas slogan décoratif |
| Ton | Franc, compact |
| Importance visuelle | Forte lorsqu’elle apparaît ; usage rare |
| Taille relative | Grande (H1 ou titre de campagne) ; jamais microcopie |
| Relation au nom | Complète Flippin’ Maple ; ne le remplace pas |
| Relation au logo | **Distincte** ; jamais soudée automatiquement |
| Relation aux titres secondaires | Les titres secondaires **complètent** ; ils ne paraphrasent pas la signature |
| Hero | Emplacement de travail possible comme H1 (**Provisoire**) |
| Campagnes / vêtements / éditorial | Usages forts possibles |
| FR / EN | Forme **anglaise conservée** dans les versions française et anglaise (**Validé**) ; **aucune traduction** (automatique ou manuelle de la signature) |
| Fréquence | Rare et intentionnelle |
| Fréquence maximale | Un usage dominant par page / par pièce visuelle forte |
| Juridique | **Ouvert** — validation avant exploitation commerciale finale |

**Usages possibles :** Hero, campagne, étiquette, visuel éditorial, vêtement, page manifeste.

**Usages à éviter :** chaque bouton, chaque courriel, chaque fiche produit, erreurs, politiques, footer répété, slogans secondaires concurrents, microcopie, plusieurs occurrences de même importance sur une page.

Ne pas expliquer littéralement la signature après chaque usage.

## 12. Architecture des messages

1. idée principale ;
2. preuve ou précision ;
3. action claire.

| Contexte | Variation |
|---|---|
| Hero | 1 + action (preuve visuelle) |
| Produit | 1 + détails factuels + CTA |
| Campagne | 1 évocateur + CTA |
| Formulaire | 1 littéral + aide |
| Courriel | 1 + détail utile + CTA |
| Service | 1 (fait) + 2 (action) + 3 (aide) |

Éviter les longs paragraphes quand un message court suffit.

## 18. Vocabulaire interdit ou non vérifié

Sans preuve vérifiable, **interdire** notamment :

- fabriqué au Canada / fait au Québec / fait à la main / artisanal ;
- écoresponsable / durable / éthique / local (comme label) ;
- premium / qualité supérieure / meilleure qualité ;
- conçu pour la performance / testé par des athlètes ;
- communauté mondiale / mouvement mondial ;
- édition limitée (si faux).

Toute affirmation factuelle doit être vérifiable. Le POD n’est pas une histoire artisanale.

## 21. Humour

Rare, sec, léger, intelligent, non forcé, jamais aux dépens du client.

**Interdit** dans : erreurs de paiement, retards, remboursements, problèmes de commande, sécurité, politiques, situations sensibles.

## 22. Microcopie d’interface

Clarté fonctionnelle **avant** personnalité. Exemples = formulations de travail, **pas** des textes finaux approuvés.

| Situation | Principe / exemple de travail | Statut |
|---|---|---|
| Navigation | Libellés courts et littéraux (`Boutique`, `Collections`). | Recommandé |
| Recherche | Placeholder : `Rechercher` ; résultats vides : `Aucun résultat.` | Provisoire |
| Compte | `Connexion` / `Créer un compte` / `Mon compte` — sans jargon. | Recommandé |
| Panier vide | `Ton panier est vide.` + CTA vers la boutique. | Provisoire |
| Panier avec produits | Totaux et lignes clairs ; pas de ton campagne. | Recommandé |
| Stock | `Rupture de stock` ; pas de fausse rareté. | Recommandé |
| Sélection de taille | `Choisir une taille` avant ajout si requis. | Recommandé |
| Quantité | Contrôles numériques simples ; libellé `Quantité`. | Recommandé |
| Ajout au panier | Confirmation courte : `Ajouté au panier`. | Recommandé |
| Erreur | Dire le problème + la correction possible ; ne pas blâmer. | Recommandé |
| Chargement | `Chargement…` — neutre, sans humour. | Recommandé |
| Succès | `Commande confirmée.` / confirmation d’action factuelle. | Recommandé |
| Champ de formulaire | Aide courte près du champ ; erreur : `Entre une adresse courriel valide.` | Provisoire |
| Inscription courriel | `S’inscrire` + promesse honnête (pas de privilèges fictifs). | Recommandé |
| Confirmation | Récapitulatif clair de l’action réussie. | Recommandé |

## 23. Pages produits

Structure verbale recommandée :

1. nom ;
2. description courte ;

4. coupe ;

6. entretien ;
7. production et expédition réelles ;
8. disponibilité ;

**Interdit :** fausses histoires de création, descriptions clones, adjectifs non prouvés, specs inventées, origine inventée, rareté artificielle.

## 24. Collections et campagnes

Une collection repose sur une idée réelle : matière, saison, mouvement, lieu, contraste, collaboration, graphique, événement.

Pas de capsule sans concept crédible.

Campagne : titre → phrase d’appui → preuve (visuelle/produit) → CTA.

## 26. Service client

Voix : claire, respectueuse, responsable, calme, précise, orientée résolution.

Les formulations ci-dessous sont génériques. **Les politiques réelles priment** ; ne pas inventer délais, remboursements garantis, responsabilités ou procédures non documentées.

| Situation | Première réponse | Information requise | Action ou solution | Ton | À éviter |
|---|---|---|---|---|---|
| Retard | Accuser réception et confirmer qu’on regarde le dossier | N° de commande, adresse, détails reçus | Suivre le statut réel et informer selon la politique | Calme | Inventer un délai |
| Erreur de commande | Reconnaître l’écart signalé | N° de commande, article attendu vs reçu | Corriger selon politique (échange, ajustement, etc.) | Responsable | Blâmer le client |
| Échange | Expliquer les prochaines étapes possibles | N° de commande, article, taille/couleur | Orienter vers le processus documenté | Clair | Promettre hors politique |
| Remboursement | Confirmer la prise en charge de la demande | N° de commande, motif, preuves si requises | Appliquer la politique de remboursement réelle | Factuel | Garantir un résultat non documenté |
| Produit endommagé | Exprimer de la considération sans dramatiser | Photos, n° de commande, description | Solution selon politique (remplacement, etc.) | Empathique calme | Humour |
| Mauvaise taille | Orienter vers échange / guide des tailles | Taille reçue, taille souhaitée, n° de commande | Selon politique d’échange | Utile | Juger le choix |
| Commande introuvable | Vérifier les identifiants fournis | Courriel, n° éventuel, date approximative | Retrouver ou expliquer l’absence selon les données | Patient | Accuser de mauvaise foi |
| Problème de paiement | Séparer technique et compte | Dernière étape atteinte, message d’erreur | Réessayer / autre moyen / support paiement selon cas | Littéral | Ton campagne, blâme |

**Éviter aussi :** jargon, promesse impossible, répétition de `NO FIXED LINE.`.

## 27. Erreurs et situations sensibles

Ne jamais : culpabiliser ; plaisanter ; inventer la cause ; promettre une résolution immédiate ; masquer l’erreur sous un message de campagne.

| Situation | Ce qui s’est passé | Action proposée | Aide disponible | Ton | Interdit |
|---|---|---|---|---|---|
| Paiement refusé | Le paiement n’a pas abouti | Réessayer ou utiliser une autre carte / méthode | Contacter le service client / banque si besoin | Calme, littéral | « Tu as fait une erreur avec ta carte. » |
| Stock épuisé | L’article n’est plus disponible | Voir d’autres pièces / revenir plus tard | Lien boutique ou similaires | Direct | Fausse rareté |
| Page introuvable | La page n’existe pas ou n’est plus là | Retour à la boutique / accueil | Navigation principale | Neutre | Humour 404 forcé |
| Erreur serveur | Un problème technique empêche l’action | Réessayer plus tard | Contacter le support si ça persiste | Calme | Inventer la cause |
| Formulaire invalide | Des champs sont incomplets ou incorrects | Corriger les champs indiqués | Messages d’aide près des champs | Utile | Blâme |
| Connexion échouée | La connexion n’a pas fonctionné | Vérifier identifiants / réessayer | Réinitialisation si disponible | Factuel | Familiarité excessive |
| Commande retardée | L’expédition ou la livraison est en retard | Consulter le suivi ; contacter le support | Service client | Responsable | Promesse de date inventée |
| Remboursement | Une demande de remboursement est en cours ou refusée selon politique | Expliquer le statut réel | Service client / politique | Factuel | Garantie non documentée |

## 28. Réseaux sociaux

- **Fréquence du ton de campagne :** occasionnelle, jamais permanente ; le quotidien reste sobre.
- **Longueur des légendes :** courtes ; une idée principale.
- **Hashtags :** rares et pertinents ; jamais en masse.
- **CTA :** clairs (`Voir la boutique`, lien en bio) ; non mystérieux.
- **Réponses aux commentaires :** calmes, utiles, même registre que la marque.
- **Modération :** retirer abus et spam ; ne pas inventer d’engagement.
- **Gestion des critiques :** factuelle, respectueuse ; orienter vers le service client si besoin.
- **Collaborations :** réelles et cohérentes avec la marque ; pas de partenariat inventé.
- **Contenu créé par les clients :** possible seulement avec **autorisation** avant réutilisation.
- **Communauté :** distinction nette entre communauté réelle et communauté inventée ; aucun volume inventé.
- **Cadence de publication :** **Ouverte** (non figée ici).
- **Tendances :** ne dictent pas la voix.

**Interdit :** hashtags en masse, slang forcé, ton changeant, faux témoignages, fausse communauté, trends forcées.

## 30. Matrice de contrôle rédactionnel

Avant publication :

- [ ] Est-il vrai ?
- [ ] Est-il vérifiable ?
- [ ] Est-il clair dès la première lecture ?
- [ ] Est-il utile ?
- [ ] Ton calme et assuré ?
- [ ] Sans slang forcé ?
- [ ] Sans clichés canadiens ?
- [ ] Sans promesses non prouvées ?
- [ ] Hiérarchie skate / surf / snowboard respectée ?
- [ ] Signature réellement nécessaire ?
- [ ] CTA clair ?
- [ ] Lisible sur mobile ?
- [ ] Adaptation naturelle (pas calque) ?
- [ ] Pourrait-il provenir de n’importe quelle marque ? → si **oui**, retravailler.

## 31. Critères de rejet

Rejeter si le texte : ressemble à un template ; est interchangeable ; invente origine / qualité / communauté / fabrication ; surutilise `NO FIXED LINE.` ; force le slang ou les métaphores ; devient touristique, patriotique, agressif ou obscur ; sacrifie la fonction ; culpabilise ; crée une fausse urgence ; traduit littéralement.

---

## 32. Critères de validation

Valider s’il : est immédiatement compréhensible ; a une personnalité reconnaissable ; reste crédible ; respecte les faits ; dit assez avec peu ; est indépendant sans arrogance ; évoque le mouvement sans cliché ; fonctionne sur mobile ; conserve la fonction commerciale ; se traduit naturellement ; n’a pas besoin du logo pour « sonner » Flippin’ Maple.
