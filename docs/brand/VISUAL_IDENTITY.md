## 1. Rôle du document

Ce fichier documente la **direction visuelle** et les **fondations de design** de Flippin’ Maple (Design Foundation v1).

Il complète :

- `docs/00_PROJECT_MASTER.md` — Pourquoi, règles fondamentales, décisions structurantes ;
- `docs/brand/VISION_AND_POSITIONING.md` — vision, positionnement, ADN, territoires émotionnels et interdits stratégiques.

Il ne crée pas un second Brand Blueprint concurrent. Il traduit le territoire stratégique en règles visuelles utilisables pour les maquettes, les placeholders et, plus tard, l’intégration.

**Non finalisé à ce stade :** le logo principal, le wordmark et le symbole. Aucun symbole temporaire ne doit être inventé pour « remplir » les maquettes.

---

## 2. Statut des décisions

### Validé

- la marque doit précéder et guider le site ;
- la direction générale doit être mature, calme, indépendante et tactile ;
- le site ne doit pas ressembler à une boutique Print-on-Demand générique ;
- le logo principal devra fonctionner en monochrome ;
- la signature verbale de travail est **« NO FIXED LINE. »** ;
- la signature doit rester distincte du logo principal ;
- le bois d’érable peut devenir une matière graphique secondaire ;
- le bois ne doit jamais créer une esthétique rustique, chalet ou boutique souvenir ;
- les images, motifs ou textures manquants devront être représentés par des placeholders descriptifs précis.

### Provisoire

- le nom de direction **« Northern Utility Editorial »** ;
- les codes hexadécimaux exacts de la palette ;
- Archivo SemiCondensed comme typographie de titres et d’interface ;
- Source Sans 3 comme typographie de texte courant ;
- les mesures précises de grille, espacements, rayons et tailles typographiques.

### Ouvert

- logo final ;
- wordmark final ;
- symbole final ;
- couleur éventuelle du logo secondaire ;
- ajustements de palette après les premières maquettes ;
- direction photographique finale ;
- actifs graphiques définitifs ;
- système de motifs complet ;
- validation juridique de **« NO FIXED LINE. »**.

## 3. Direction créative

### Territoire de travail (Provisoire)

**Northern Utility Editorial**

Combinaison de :

- streetwear mature ;
- culture skate comme origine principale, avec des influences complémentaires du surf et du snowboard ;
- fonctionnalité workwear ;
- environnement canadien froid et urbain ;
- matière naturelle ;
- compositions éditoriales calmes ;
- robustesse ;
- retenue ;
- mouvement maîtrisé.

### Principe directeur

> Une structure solide que la marque choisit parfois de traverser.

La structure (grille, typographie, surfaces) reste lisible et stable. La marque peut la franchir avec intention — jamais par bruit gratuit.

### Ce que la marque ne doit pas devenir

- corporative ;
- technologique ;
- esport ;
- crypto ;
- touristique ;
- rustique ;
- faussement luxueuse ;
- artificiellement rebelle ;
- adolescente ou caricaturale.

Ces interdits rejoignent le positionnement de `VISION_AND_POSITIONING.md` (skate mature, Canada discret, pas de faux premium).

## 4. Palette de travail

**Statut : Provisoire** — à ajuster après les premières maquettes et tests de contraste.

| Nom | Hex | Rôle | Usages autorisés | Usages à éviter |
|---|---|---|---|---|
| **Ink Black** | `#11110F` | Texte et structure sombres | Texte principal, logo monochrome, boutons principaux, séparateurs | Éviter les grandes surfaces sombres monotones sans respiration ; ne pas utiliser Ink Black pour remplacer systématiquement Deep Forest dans son rôle d’accent identitaire |
| **Maple Bone** | `#F1ECE2` | Fond clair principal | Surfaces claires, texte sur fonds sombres, alternative chaude au blanc pur | Blanc pur comme fond par défaut ; teinte « crème souvenir » saturée |
| **Charcoal** | `#292A27` | Surfaces sombres secondaires | Footer, sections sombres, blocs secondaires | Remplacer Ink Black pour le texte long sur fond clair |
| **Deep Forest** | `#304238` | Accent identitaire | Fonds choisis, certains états actifs ou CTA | Couleur dominante permanente ; confondre avec « green outdoor générique » |
| **Weathered Maple** | `#B6A184` | Matière secondaire | Motif de bois, détails éditoriaux, fonds ponctuels | Esthétique chalet / cabane ; fond global saturé |
| **Cold Concrete** | `#737772` | Neutre fonctionnel | Texte secondaire non critique, métadonnées, éléments fonctionnels ou décoratifs | Texte principal ; bas contraste sur Maple Bone sans vérification |
| **Oxide Red** | `#893B32` | Accent rare | Campagnes ou capsules ponctuelles | Couleur promotionnelle permanente ; boutons d’urgence ; soldes « flash » |

### Hiérarchie indicative (non rigide)

- ~60 % Maple Bone ;
- ~25 % Ink Black et Charcoal ;
- ~10 % Deep Forest ;
- ~5 % Weathered Maple et Cold Concrete ;
- Oxide Red uniquement de manière exceptionnelle.

Cette répartition est un guide de proportion, pas une formule à appliquer page par page.

### Combinaisons prioritaires à tester

## 6. Typographie d’interface

**Statut : Provisoire**

| Rôle | Famille |
|---|---|
| Titres, navigation, éléments de caractère | **Archivo SemiCondensed** |
| Texte courant et interface | **Source Sans 3** |

### Usages proposés

**Archivo SemiCondensed**

- titres ;
- navigation ;
- collections ;
- prix importants ;
- appels éditoriaux ;
- certains libellés courts.

**Source Sans 3**

- paragraphes ;
- descriptions de produits ;
- formulaires ;
- boutons ;
- panier ;
- checkout ;
- politiques ;
- informations techniques.

### Poids recommandés

**Archivo SemiCondensed :** 600 · 700 · 800 (avec retenue)

**Source Sans 3 :** 400 · 500 · 600

### Règles

- capitales réservées aux textes courts ;
- aucune capitale intégrale dans les longs paragraphes ;
- pas plus de deux familles typographiques ;
- pas de fonte condensée pour les longs textes ;
- priorité constante à la lisibilité.

## 10. Formes, bordures et ombres

**Statut : Provisoire (formes) / Validé (principe de hiérarchie)**

- rayons principalement entre 2 et 4 px ;
- rayon confortable maximal autour de 8 px ;
- pas de boutons en pilule ;
- pas de grandes cartes très arrondies ;
- images généralement rectangulaires ;
- bordures fines de 1 px ;
- séparateurs droits et sobres ;
- presque aucune ombre.

La hiérarchie doit principalement venir :

- de l’espace ;
- du contraste ;
- de la typographie ;
- des changements de fond ;
- de la photographie ;
- de la matière.

**Statut : Provisoire**

### Bouton principal clair

- fond Ink Black ;

### Bouton secondaire

- fond transparent ;

- texte Ink Black ;

### Sur fond sombre

- fond Maple Bone ;

### Interdits

- dégradés ;
- ombres ;

- texte vague comme « Cliquez ici » ;

## 13. Motif inspiré du bois d’érable

**Statut : Validé (principe) · Provisoire (versions / opacités)**

Le bois est un **actif secondaire**. Il ne définit pas seul la marque.

### Le motif doit être

- abstrait ;
- inspiré du fil naturel de l’érable ;
- composé de lignes organiques longues et irrégulières ;
- monochrome ;
- peu contrasté ;
- seamless lorsque nécessaire ;
- compatible avec le site, les emballages et les applications imprimées.

### Il ne doit pas présenter

- de planches visibles ;
- de nœuds rustiques ;
- de teinte orangée excessive ;
- d’esthétique chalet ;
- d’esthétique cabane à sucre ;
- d’effet faux bois numérique dominant.

### Versions provisoires

- Ink Black à environ 4–6 % sur Maple Bone ;
- Maple Bone à environ 5–8 % sur Charcoal ou Deep Forest.

### Usages possibles

- sections éditoriales ;
- manifeste ;
- emballage ;
- transitions de collections ;
- imprimés ;
- arrière-plans ponctuels.

### Ne pas l’utiliser

- sur toutes les cartes produits ;
- comme fond global permanent ;
- derrière de petits textes ;
- derrière le logo en permanence ;
- lorsqu’il réduit la lisibilité.

## 14. Règle des placeholders visuels

**Statut : Validé**

Lorsqu’une maquette ou une spécification prévoit une image, une image de fond, une texture, un motif ou un autre actif qui n’existe pas encore, un **placeholder descriptif** doit être placé exactement à l’endroit prévu.

Chaque placeholder doit préciser au besoin :

- rôle ;
- sujet ;
- direction artistique ;
- cadrage ;
- espace négatif ;
- palette ;
- ratio ;
- dimensions minimales ;
- format cible ;
- niveau de contraste ;
- contexte d’utilisation ;
- éléments interdits ;
- comportement responsive ;
- texte alternatif attendu.

### Exemple 1 — Placeholder Hero

```
[PLACEHOLDER — IMAGE HERO]

Rôle : bandeau d’accueil, premier plan visuel
Sujet : silhouette urbaine froide, vêtement Flippin’ Maple porté naturellement
Direction : lumière naturelle, ciel couvert, béton / asphalte, pose non théâtrale
Cadrage : horizontal éditorial, sujet décentré
Espace négatif : zone claire à gauche ou à droite pour titre + CTA
Palette : tons Maple Bone / Charcoal / Deep Forest ; pas de saturation touristique
Ratio : 16:9 (desktop) ; recadrage 4:5 prévu mobile
Dimensions minimales : 2400 × 1350 px (desktop)
Format cible : WebP / JPEG haute qualité
Contraste : moyen ; texte overlay lisible sur zone réservée
Contexte : page d’accueil — premier viewport
Interdits : mockup Printful dominant, logos tiers, néons, sourire publicitaire
Responsive : conserver le sujet principal ; rogner les bords, pas le visage / produit clé
Alt attendu : description factuelle du sujet et du lieu, sans slogan
```

### Exemple 2 — Placeholder motif veinage d’érable

```
[PLACEHOLDER — MOTIF VEINAGE ÉRABLE]

Rôle : texture de fond secondaire, section éditoriale
Sujet : fil abstrait inspiré de l’érable ; lignes organiques longues et irrégulières
Direction : monochrome, peu contrasté, seamless
Cadrage : motif répétitif ; aucun nœud rustique visible
Palette : Ink Black ~4–6 % sur Maple Bone (ou Maple Bone ~5–8 % sur Deep Forest)
Ratio : tuile carrée 1:1 ou 2:1 selon usage
Dimensions minimales : 1024 × 1024 px (tuile)
Format cible : SVG ou PNG transparent
Contraste : faible ; ne jamais nuire au texte
Contexte : manifeste, transition de collection, emballage — usage ponctuel
Interdits : planches, chalet, cabane à sucre, teinte orangée, faux bois 3D
Responsive : conserver l’opacité ; ne pas densifier le motif sur petit écran
Alt attendu :
- si le motif est purement décoratif : `alt=""` lorsqu’il s’agit d’une image HTML, ou aucune alternative textuelle s’il est appliqué en arrière-plan CSS ;
- ne fournir une description textuelle que si le motif transmet exceptionnellement une information utile au contenu.
```

## 15. Mouvement et interactions

**Statut : Provisoire**

- durée standard approximative : 150 à 250 ms ;
- transitions discrètes ;
- changements de contraste ou déplacement léger ;
- zoom très faible sur certaines images ;
- apparition sobre ;
- prise en charge de `prefers-reduced-motion`.

### Interdits

- glitch ;
- rotations gratuites ;
- parallaxe agressive ;
- curseur personnalisé ;
- titres constamment animés ;
- motif de bois animé en permanence ;
- interactions qui ralentissent l’achat.

---

## 16. Principes responsive

**Statut : Validé (principes) · Provisoire (mesures associées)**

- priorité à la lisibilité ;
- hiérarchie préservée sur mobile ;
- images recadrées intentionnellement ;
- boutons suffisamment grands ;
- aucun texte essentiel intégré directement dans une image ;
- aucun comportement desktop simplement compressé ;
- maintien des marges et espaces respirants ;
- tests sur petits écrans obligatoires.

---

## 17. Critères de cohérence

Avant chaque décision visuelle importante :

- Est-ce distinctif ?
- Est-ce lisible ?
- Est-ce crédible pour une vraie marque de vêtements ?
- Est-ce mature ?
- Est-ce tactile sans devenir rustique ?
- Est-ce canadien sans devenir touristique ?
- Est-ce inspiré des board sports sans devenir caricatural ?
- Est-ce cohérent avec « NO FIXED LINE. » ?
- Est-ce compatible avec un logo monochrome ?
- Est-ce que cela ressemble trop à un template e-commerce ?
- Est-ce que cela renforce la marque ou ajoute seulement de la décoration ?
