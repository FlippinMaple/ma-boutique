## 1. Rôle du document

Ce fichier est la **source officielle détaillée** pour :

- la vision de marque ;
- le positionnement ;
- l’ADN ;
- le territoire émotionnel ;
- le public ;
- les critères stratégiques qui guideront les décisions créatives et produit.

Il développe ce que la Constitution pose comme cap, sans la remplacer et sans la recopier inutilement. La Constitution répond au **Pourquoi** et fixe les règles non négociables. Ici, on répond au **Quoi** de la marque : ce qu’elle veut devenir, pour qui, avec quels repères et quelles limites.

Toute décision de voix, d’identité visuelle, de collection ou d’évolution du site devra pouvoir se justifier à partir de ce document — ou d’une mise à jour explicite de celui-ci.

## 2. Ambition de marque

Flippin’ Maple ne doit pas rester une simple boutique de vêtements imprimés à la demande.

L’ambition est de construire progressivement une **marque indépendante**, cohérente, reconnaissable et durable. Pas un catalogue interchangeable. Pas un storefront technique. Une marque que l’on peut préférer.

Le client ne doit pas seulement penser :

> « J’aime ce chandail. »

Il devrait progressivement penser :

> « J’aime Flippin’ Maple. »

Le but est de développer une **préférence de marque**, pas seulement de vendre des impressions sur des vêtements. La vente reste nécessaire ; elle n’est pas suffisante. Sans préférence, chaque commande repart à zéro. Avec préférence, chaque produit renforce un actif.

---

## 3. Vision à long terme

Sur plusieurs années, Flippin’ Maple doit pouvoir devenir une marque crédible dans des contextes très différents, sans changer de personnalité à chaque fois :

- une boutique indépendante ;
- un skate shop ;
- une sélection streetwear ;
- une campagne photo ;
- une étiquette brodée ;
- une casquette ;
- une planche ;
- un autocollant ;
- l’esprit de clients capables de reconnaître son univers et son symbole.

La marque doit pouvoir évoluer — formats, canaux, niveaux de contrôle de production — **sans devoir être entièrement reconstruite** chaque fois que les tendances changent. Les tendances peuvent informer. Elles ne doivent pas dicter l’identité.

Cette vision s’aligne sur l’ambition premium et internationale de la Constitution, tout en ancrant le trajet dans un territoire culturel clair et tenable.

## 4. Positionnement officiel

Flippin’ Maple est une **marque canadienne indépendante de vêtements et de streetwear**, issue de la culture skate, mais accessible au-delà des skateurs.

### Le skate, tel qu’il doit être utilisé

Le skate est :

- une **origine culturelle** ;
- une **attitude** ;
- une **influence** ;
- un rapport au mouvement, à la liberté et à l’indépendance.

### Le skate, tel qu’il ne doit pas être utilisé

Le skate ne doit jamais devenir :

- un costume ;
- une caricature ;
- une obligation pour le client ;
- un prétexte pour empiler des symboles clichés.

Une personne **ne doit pas avoir besoin de faire du skateboard** pour porter Flippin’ Maple. L’appartenance se fait par affinité d’univers, pas par preuve de pratique.

---

## 5. Promesse de marque

**Promesse stratégique (version de travail durable) :**

Flippin’ Maple propose des vêtements intentionnels, portés par une identité forte mais calme — culture canadienne discrète, influence skate mature, sélection maîtrisée — construits pour être crédibles comme marque, pas comme catalogue.

Cette promesse s’appuie sur :

- des vêtements intentionnels ;
- une identité forte mais calme ;
- une culture canadienne discrète ;
- une influence skate mature ;
- une sélection maîtrisée ;
- une marque construite avec cohérence.

Elle refuse les superlatifs vides et les promesses de qualité non vérifiées. « Premium » engage ; il ne s’invente pas dans un slogan.

---

## 6. Origine culturelle

Trois sources nourrissent l’univers. Aucune ne doit écraser les autres.

### 6.1 Culture skate

Références utiles :

- indépendance ;
- mouvement ;
- appropriation des espaces ;
- créativité ;
- progression ;
- confiance ;
- résistance aux conventions.

À éviter : le skate adolescent caricatural, les symboles ajoutés artificiellement, le « look skater » sans fond.

### 6.2 Identité canadienne

Présenter une identité canadienne :

- authentique ;
- discrète ;
- contemporaine ;
- non touristique ;
- non folklorique.

Elle peut être évoquée par :

- le climat ;
- les matières ;
- le territoire ;
- la retenue ;
- la robustesse ;
- la feuille d’érable lorsqu’elle est traitée avec originalité.

Elle ne doit pas reposer sur :

- les castors ;
- les orignaux ;
- les bûcherons caricaturaux ;
- les chemises à carreaux comme raccourci ;
- la feuille d’érable rouge générique ;
- l’esthétique de boutique souvenir.

### 6.3 Culture indépendante

Flippin’ Maple se construit progressivement, honnêtement, avec contrôle. Indépendante ne veut pas dire improvisée. Cela veut dire : direction claire, décisions assumées, rythme réaliste.

Le Print-on-Demand est un **moyen de production initial**.
Il ne constitue **ni l’identité ni la proposition de valeur** de la marque.

## 7. ADN de marque

| Qualité | Sens concret |
|---|---|
| **Authentique** | Dire ce qui est vrai aujourd’hui ; ne pas fabriquer un héritage. |
| **Indépendante** | Choisir sa direction plutôt que suivre le catalogue fournisseur. |
| **Confiante** | Affirmer sans crier ; laisser le travail parler. |
| **Calme** | Force par la retenue, pas par le bruit. |
| **Simple** | Aller à l’essentiel ; refuser le superflu décoratif. |
| **Solide** | Visuels, produits et récit tenables dans le temps. |
| **Créative** | Originalité maîtrisée, pas invention permanente pour remplir. |
| **Légèrement rebelle** | Distance critique face aux conventions, sans posture agressive. |
| **Accessible** | Porte ouverte au-delà du cercle skateur pur. |
| **Premium sans prétention** | Exigence de présentation et de cohérence, sans faux luxe. |
| **Canadienne sans être touristique** | Ancrage culturel discret, jamais souvenir. |
| **Inspirée du skate sans être adolescente** | Attitude mature, pas costume de scène. |
| **Intemporelle plutôt que dépendante des tendances** | Évolution possible sans reset d’identité. |

## 9. Public de marque

La cible n’est **pas** réduite à des hommes skateurs de 18 à 25 ans.

Flippin’ Maple s’adresse à des adultes et jeunes adultes attirés par :

- le streetwear sobre ;
- la culture skate ;
- les vêtements graphiques maîtrisés ;
- le workwear ;
- les marques indépendantes ;
- l’identité canadienne discrète ;
- les objets bien conçus ;
- les vêtements polyvalents ;
- une esthétique urbaine authentique.

Les profils détaillés (segments, motivations, usages) devront être **validés plus tard** par la recherche et l’observation réelle. Ce document n’invente ni statistiques, ni revenus, ni comportements non vérifiés.

## 10. Relation souhaitée avec le client

La marque cherche à créer, dans le temps :

- reconnaissance ;
- confiance ;
- attachement ;
- préférence ;
- sentiment d’alignement culturel ;
- fidélité à l’univers plutôt qu’à un seul produit.

## 16. Critères décisionnels

Pour toute décision importante (visuel, produit, copy, partenariat, évolution de site), poser :

1. Est-ce distinctif ?
2. Est-ce cohérent ?
3. Est-ce crédible ?
4. Est-ce portable ?
5. Est-ce productible ?
6. Est-ce mémorable ?
7. Est-ce durable ?
8. Est-ce réellement Flippin’ Maple ?
9. Est-ce qu’une marque indépendante sérieuse ferait ce choix ?
10. Est-ce que cela ressemble trop à une marque existante ?
11. Est-ce que cela renforce la marque ou seulement le visuel ?
12. Est-ce que cela paraît authentique ou artificiellement cool ?
13. Est-ce que cela augmente ou diminue la qualité perçue ?

Une réponse négative claire à plusieurs de ces questions suffit à refuser ou à retravailler.

## 17. Trajectoire d’évolution

Progression volontaire :

1. clarifier le positionnement ;
2. établir la direction créative ;
3. développer le système d’identité ;
4. créer des produits représentatifs ;
5. améliorer la présentation ;
6. augmenter progressivement le contrôle sur les vêtements, les techniques d’impression, les étiquettes et l’emballage ;
7. envisager plus tard l’inventaire propre, les séries limitées ou une production plus spécialisée.

Aucune origine, qualité de fabrication ou exclusivité ne doit être inventée pour « paraître plus loin » que la réalité actuelle.

---

## 18. Règles de vérité

- ne pas prétendre à une fabrication artisanale inexistante ;
- ne pas inventer un héritage de plusieurs décennies ;
- ne pas prétendre avoir une immense communauté ;
- ne pas utiliser une fausse rareté ;
- ne pas promettre une qualité non vérifiée ;
- distinguer l’ambition future de la réalité actuelle ;
- présenter le Print-on-Demand honnêtement sans le laisser définir la marque.

## 19. Langue de marque

Direction linguistique **validée** (détail rédactionnel dans `docs/brand/VOICE.md`) :

- le **français québécois naturel** est la langue source et principale ;
- le **tutoiement** est le registre principal (direct, mature, accessible, cohérent) ;
- l’**anglais canadien** est offert comme version complète, adaptée naturellement — jamais une traduction littérale ;
- **une seule langue** est affichée à la fois dans les contenus principaux ;
- **`NO FIXED LINE.`** demeure en anglais dans les versions française et anglaise ; il ne se traduit pas.

Le sélecteur de langue, la mémorisation de préférence et l’architecture technique bilingue relèvent de l’intégration future, hors portée de ce document.

## 21. Checklist de validation stratégique

Avant d’adopter une proposition (visuelle, produit, message, offre) :

- [ ] Cohérente avec le positionnement officiel
- [ ] Sans cliché canadien, skate ou streetwear caricatural
- [ ] Crédible comme marque indépendante
- [ ] Accessible au-delà des skateurs
- [ ] Identité canadienne discrète (pas touristique)
- [ ] Durable dans le temps (pas seulement « tendance »)
- [ ] Honnête par rapport à la réalité actuelle
- [ ] Productible dans les moyens actuels ou clairement phaseable
- [ ] Distinctive face aux concurrentes / références
- [ ] Renforce une préférence de marque, pas seulement un visuel isolé

## 22. Statut des décisions

- le nom **Flippin’ Maple** est conservé sauf problème stratégique majeur démontré ;
